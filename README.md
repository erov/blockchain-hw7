# Exam tickets distribution

A simple C++ project that allows to generate ticket numbers for specified list of students. Supports specifying `salt` parameter for changing distribution.
At the ending of stdout there is a little statistic about number of students with specified ticket number.

## Preparing
```
$ git clone git@github.com:System-Glitch/SHA256.git 
```

## Usage 
```
$ hw7 --filename <filename> --tickets <number of tickets> --salt <int64_t nuber>
```
